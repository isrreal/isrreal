## Selected Public Projects

### 📐 [Triple Roman Domination in Graphs](https://github.com/isrreal/Triple-Roman-Domination-in-graphs)

> *Graph Theory · Combinatorial Optimization · Metaheuristics · C++*

Algorithms for the **Triple Roman Domination Problem**, an NP-complete combinatorial optimization problem defined over graphs.

I implemented and experimentally evaluated multiple optimization approaches, including a Genetic Algorithm and a MAX-MIN Ant System combined with Random Variable Neighborhood Search. An Integer Linear Programming formulation was used as an exact baseline for solution-quality validation.

* **Mathematical focus:** Graph theory, discrete optimization and integer programming
* **Algorithms:** Genetic Algorithm, Ant Colony Optimization, MAX-MIN Ant System and RVNS
* **Evaluation:** Heuristic solution quality compared against exact optimization results
* **Stack:** C++ · Bash · Linux · Integer Linear Programming

### 🕐 [Face Clock Evoluir — Facial Recognition Attendance System](https://github.com/isrreal/face-clock-evoluir-public)

> *Computer Vision · Applied Machine Learning · Backend Engineering · Production System*

Public technical case study of a production attendance system developed for an educational institution.

The system combines facial recognition, vector similarity search, GPS-based geofencing and automated medical-certificate classification within a secure backend architecture.

* **Computer vision:** 1:1 verification and 1:N identification with 512-dimensional facial embeddings
* **Vector search:** Cosine similarity and KNN retrieval using PostgreSQL and `pgvector`
* **Robustness:** Multi-frame aggregation, liveness validation and replay protection
* **Document classification:** EfficientNet-B0 combined with an OCR-based decision pipeline
* **Engineering:** FastAPI, JWT revocation, SQLAlchemy, Alembic, Docker and GitHub Actions
* **Quality:** 115 automated tests and continuous integration

> The production source code, biometric data, operational configuration and identifiable client information remain private.

### 🦟 [End-to-End Dengue Forecasting System](https://github.com/isrreal/Dengue-Forecasting-System)

> *Time Series · Multitask Learning · MLOps · Public Health Data*

End-to-end platform for forecasting dengue notifications using public epidemiological data from Brazil's SINAN system.

The project compares LSTM and Progressive Layered Extraction architectures within a reproducible MLOps pipeline covering data extraction, model training, experiment tracking, serving and visualization.

* **Key result:** PLE reduced validation RMSE from approximately **98 to 83** on the Ceará time series
* **Models:** LSTM and Progressive Layered Extraction
* **Data pipeline:** SINAN extraction and transformation using `pySUS`
* **MLOps:** MLflow experiment tracking, containerized services and automated tests
* **Serving:** FastAPI REST API and interactive dashboard
* **Stack:** PyTorch · MLflow · FastAPI · PostgreSQL · Docker Compose · Streamlit

### 🇧🇷 [Big Data Mining — Brazilian Emergency Aid](https://github.com/isrreal/AuxilioEmergencialQueries)

> *Data Engineering · Database Optimization · High-Volume ETL*

Data-engineering pipeline for exploring approximately **257 million Brazilian government records** related to emergency-aid payments.

The project focuses on scalable ingestion, asynchronous processing, SQL query optimization and interactive exploration of large public datasets.

* **Data volume:** Approximately 257 million records
